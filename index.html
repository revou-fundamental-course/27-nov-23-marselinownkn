Luas Segitiga

Rumus Luas Segitiga yaitu :

L = 1/2 x a x t

Di mana :
L = Luas
a = Panjang Atas
t = Tinggi

Keliling Segitiga

Rumus Keliling Segitiga yaitu :

K = S1 + S2 + S3

Di mana :
K = Sisi A
S1 = Sisi B
S3 = Sisi C
